Unity Kata #4 - Coin flip

Flow:
 - select heads or tails
 - push coin off a bar
 - coin lands on table, heads or tails is revealed
 - global wins or losses is increased
 - play again
 
Requirements:
 - UI for selecting heads or tails
 - table, coin, bar model (cubes :D)
 - push mechanic
 - physics on coin
 - PlayerPrefs: win_count, lose_count, current_bet
 - UI for results and playing again
